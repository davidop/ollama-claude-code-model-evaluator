# Roadmap Issue 01: English-first discoverability and launch channels

## Problem

The project has strong technical value, but discoverability is constrained if launch messaging and docs are not consistently optimized for global audiences and social channels.

## Proposal

- Keep Spanish + English documentation parity over time
- Add short launch snippets for X, Reddit and Hacker News in docs
- Add one benchmark snapshot block designed for copy-paste sharing
- Define one canonical project tagline and value proposition

## Expected Impact

- Higher conversion from visitors to stars
- Better traction in international communities
- Easier community sharing of reproducible results

## Acceptance Criteria

- README and README.en stay synchronized for key sections
- docs include reusable launch text for 3 channels
- a short benchmark snapshot template exists and is updated per release

## Labels

enhancement, growth, docs
